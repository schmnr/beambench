# Support

GitHub issues are for reproducible source-code bugs and actionable feature
proposals.

For installation help, settings questions, connection troubleshooting, or
diagnostic reports, use [Beam Bench Support](https://beambench.com/support).
The application can submit a diagnostic report from **Help > Report a Bug** and
can inspect connection problems under **Tools > Connection Diagnostics**.

For security vulnerabilities, follow [SECURITY.md](SECURITY.md) instead of
posting publicly.
